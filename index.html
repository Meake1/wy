访问受限

由于域名备案问题，直接访问可能受限。请使用代理工具访问我们的网站。

推荐代理工具

Clash

跨平台代理工具

功能强大的代理客户端，支持多种协议，界面友好。

V2RayN

Windows 专用客户端

轻量级Windows代理工具，配置简单，性能优秀。

配置教程

Clash 配置步骤

1

下载并安装 Clash

从官网下载对应系统的 Clash 客户端并安装。

2

导入配置文件

将以下配置保存为 config.yaml 文件：

port: 7890 socks-port: 7891 allow-lan: true mode: Rule log-level: info external-controller: :9090 proxies: - name: "MyProxy" type: http server: your-proxy-server port: 8080 username: your-username password: your-password proxy-groups: - name: "Proxy" type: select proxies: - "MyProxy" - DIRECT rules: - DOMAIN-SUFFIX,your-domain.com,Proxy - MATCH,DIRECT
3
\

启动代理

点击启动按钮，Clash 会自动加载配置并开始代理。

V2RayN 配置步骤

1

下载 V2RayN

下载最新版本的 V2RayN 压缩包并解压。

2

添加服务器配置

点击服务器 -> 添加VMess服务器，填写以下信息：

地址：your-proxy-server 端口：8080 用户ID：your-uuid 额外ID：0 加密方式：auto 传输协议：tcp
3

启用系统代理

右键点击托盘图标，选择启用系统代理。

浏览器代理设置

Chrome

安装 SwitchyOmega 扩展

Firefox

设置 -> 网络设置

Safari

偏好设置 -> 网络

等待代理连接...

配置完成后，请刷新此页面继续访问
